用户管理命令

本节介绍如下几个用户管理命令：

  • useradd
  • passwd
  • who
  • w

useradd

功能：添加用户
命令所在路径： /usr/sbin/useradd（可用which或whereis查看）
由于命令在sbin里，故只有root用户有创建用户的权限。

passwd

功能：为用户修改密码。
命令格式：

注意：passwd比密码password少o和r。

who

功能：查看登陆用户命令

  • tty 本地终端
  • pts 远程终端

注意：who命令后面不需要跟任何参数

w

功能：得到比who更加详细的用户登陆信息命令
下面介绍提供的登录信息中的几个参数：

  • up 表示系统连续运行了多久
  • load average 表示平均负载情况
  • IDLE 用户空闲时间。一旦执行任何操作，计时器会清零重新计时。
  • JCPU 和该终端连接的所有进程占用的时间。
  • PCPU 当前进程占用时间。
  • WHAT 当前运行进程的命令行操作

压缩解压指令

压缩解压指令是在Linux中很常用的一类指令。
对于zip格式，win和linux中均可使用。
但rar格式的压缩包在Linux中需要安装rar工具才能解压缩。
一般Linux的压缩包在win中都能打开，但win中的压缩格式Linux基本无法打开。

本节中介绍的命令有如下几个：

  • gzip
  • tar
  • zip
  • bzip2

gzip

gzip压缩后压缩格式为：.gz
英文原意：GNU zip

  • gzip只能压缩文件，不能压缩目录
  • gzip压缩文件后原文件消失，只剩压缩包了。

tar

命令功能：对目录进行打包，使系统可将其当作一个文件处理。
tar有几个非常常用的选项：

  • -c 打包选项
  • -x 解包选项
  • -v 显示详细信息
  • -f 指定文件名
  • -z 按照gzip格式压缩或者解压
  • -j 按照bzip2格式压缩或者解压

解压目录不需要写解压后的文件名。

zip

zip与gzip同为压缩命令，与gzip相比有如下特点：

  • 压缩后保留原文件
  • 加入-r选项后可以压缩目录
  • 压缩比相较gzip差很多。

开机关机命令

shutdown

shutdown关机命令有如下几个常用参数：

  • -h 关机
  • -r 重启
  • -c 取消前一个关机命令

提醒：除shutdown外，还有一些关机命令，现列举如下：

  • halt
  • poweroff
  • init 0

一些重启命令，列举如下：

  • roboot
  • init 6

我们注意到，init后面可以跟不同的数值，实际上，init代表系统运行级别，系统运行级别共有七个，列举如下：

  • init 0 关机
  • init 1 单用户模式
  • init 2 不完全多用户模式，不含NFS(network file system)服务，NFS存在安全隐患，故存在单独的运行级别。
  • init 3 完全多用户模式
  • init 4 未分配
  • init 5 图形界面
  • init 6 重启

如果我们想要更改运行级别，我们可以进入/etc/inittab下修改。
