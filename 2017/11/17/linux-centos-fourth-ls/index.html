Linux目录操作命令及帮助命令

发表于 2017-11-17 | 分类于 IT技术

目录操作命令

当命令有多个选项时，可以写在一起，和顺序无关。

目录列表命令ls

命令英文原意：list
功能描述：显示目录文件
常用选项：

  • -a 显示所有文件，包括隐藏文件（以.开头的文件即为隐藏文件，我们也可以自行创建，隐藏文件的作用并非是不让人看到，而是告诉人们，隐藏文件较为重要，没事不要胡乱删除。）
  • -l 详细信息显示，显示信息依次包括：权限位，引用次数，所有者，所属组，文件大小修改日期，文件名。
  • -lh 其中h的全称为human，就是以人类能看懂的方式显示，由于-l列出的文件大小以字节为单位，需要复杂的换算才能得到想要的结果，故-h选项应运而生。
  • -ld 查看目录属性，如果直接-l，显示的是当前目录下的文件详细信息，而加入-d后则显示当前目录的详细信息。

目录切换命令cd与目录查询命令pwd

cd英文原意为change directory
pwd英文原意为 print working directory

复制命令cp

cp英文原意为copy，默认情况下只允许复制文件，cp可以同时复制多个文件，最后一个是目标目录。

注意:cp对最后一个参数是目标还是文件很敏感，即最后是否加/，如果加/,表示将文件复制到目录中，如果不加/，表示将文件复制并改名为目标文件名。

-r选项

cp默认情况下不可复制目录，但加入-r选项后即可复制目录。

要区分cp与下面要介绍的mv，mv默认可以剪切目录，因此不需要加-r选项。但rm如果要删除目录，也必须加-r选项。

-p选项

如果我们希望复制的文件与原文件有相同的属性，可以使用-p选项，意为保存原文件或目录的属性。

剪切命令mv

该命令有两点需要说明：

  • mv命令可做剪切命令同时也可做改名命令。
  • mv默认可以剪切文件夹，不需要加任何选项。

删除命令rm

删除命令一般常用参数有如下两个：

  • -f
    其中f为force的意思，加这个参数后，删除文件时系统不在询问是否删除。
  • -r
    与cp命令一样，这个参数让rm具有了删除命令的功能，但是要注意，这个命令常与-f命令连用，否则，如果文件夹里有许多文件，该命令会一个文件一个文件来询问你是否删除。

帮助命令

当我们在Linux中遇到不懂得命令或者配置文件时，首先想到的应该是查看帮助文档而不是去百度命令格式，因为Linux自带了强大而又规范的帮助命令。

帮助命令man

man是Linux中最常用的一个帮助命令，它既可以查看命令的帮助文档，也可以查看配置文件的帮助文档。需要特别注意的是，查看配置文件帮助文档时，并不需要写清楚配置文件的绝对路径，这样反而会出错，我们只要写清楚命令名称即可。
